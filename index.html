Introduction
  • The basic syntax is same as normal markdown, such as # indicates a header, > indicates a quota, and so on.
  • Besides our online markdown editor supports row and column span markdown table.
        Use |(-m)...| to create a m-span column span table cell <td colspan="m">.
        Use |(=n)...| to create a n-span row span table cell <td rowspan="n">.
        Example: 
    |(=2)align center|align left|align right|
|a|b|
|:-:|:--|--:|
|c|d|e|
|(-2)f|g|
|h|(-2)(=3)i|
|j|
