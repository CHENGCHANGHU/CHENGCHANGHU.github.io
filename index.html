Introduction
  • The basic syntax is same as normal markdown, such as # indicates a header, > indicates a quota, and so on.
  • Basic tables are made by separating each cell with the pipe | symbol, and can be enhanced further with additional options to create advanced table layouts. This online markdown editor supports rowspan and colspan markdown table.
        Use |(-m)...| to create a m column span table cell <td colspan="m">.
        Use |(=n)...| to create a n row span table cell <td rowspan="n">.
        Example: 
    |(=2)align center|align left|align right|
|a|b|
|:-:|:--|--:|
|c|d|e|
|(-2)f|g|
|h|(-2)(=3)i|
